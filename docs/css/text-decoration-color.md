## text-decoration-color



text-decoration-color 属性用于设置元素中文本的修饰线颜色。


#### uni-app x 兼容性
| Web | Android | iOS | HarmonyOS | HarmonyOS(Vapor) |
| :- | :- | :- | :- | :- |
| 4.0 | x | x | x | x |




### 语法
```
text-decoration-color: <color>;
```



### 值限制
- color






### 适用组件 @unix-tags 
 - [text](/component/text.md)
- [button](/component/button.md)





#### App平台差异
+ App平台默认颜色为文字颜色，暂不支持设置
