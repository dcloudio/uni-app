# uni-app
`uni-app` 是一个使用 `Vue.js` 开发**跨平台**应用的前端框架。

开发者通过编写 `Vue.js` 代码，`uni-app` 将其编译到 iOS、Android、微信小程序等多个平台，保证其正确运行并达到优秀体验。
